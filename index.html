Modified Jumbotron Cover

This is a modified jumbotron that occupies the entire window. Perfect for a single splash page. The flex class centers the content vertically in the window.

CSS


#masthead {
  background-image: url("img.jpg");
  width: 100%;
  height: 100%;
  background-position: center;
  background-repeat: no-repeat;
  background-size: cover;
  color: #fafafa;
  display: flex;
  align-items: center;
  letter-spacing: .1rem;
  margin-top: 100px;;  
  }

Moving on

To the two types of Bootstrap Jumbotrons

Jumbotron

The jumbotron is used to draw extra attention to featured content or information.

HTML

<div class="container">

<div class="jumbotron my-5"> //my-5 for extra padding top and bottom

...content goes here

...content goes here

...content goes here

</div>

</div>

Large Jumbotron text is <h1 class="display-3">

The default background-color for the Jumbotron is #E9ECE9.

Click the button below to set the .jumbotron class background-color to yellow.

Fluid Jumbotron

This is a modified jumbotron that occupies the entire horizontal space of its parent.
